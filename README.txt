Anleitung – Lokaler Test + GitHub Pages

Lokaler Test (mit Python):
1. Terminal öffnen
2. In den Projektordner wechseln
3. Befehl: python -m http.server
4. Im Browser öffnen: http://localhost:8000

GitHub Pages:
1. Repository anlegen
2. Dateien hochladen
3. In Einstellungen GitHub Pages aktivieren
4. Projekt ist online

Viel Erfolg mit dem Schloss-Projekt!
